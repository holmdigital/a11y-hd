# HolmDigital Wiki

> Private knowledge base and documentation for the Holm Digital Accessibility Ecosystem.

This is a Next.js application that serves as the central documentation hub, API reference, and component playground.

**Note:** This package is `private` and is NOT published to NPM. It is deployed as a standalone site/app.

## Development

```bash
# Install dependencies (root)
npm install

# Start local dev server
npm run dev -w holmdigital-wiki
```

The site will run at [http://localhost:5173](http://localhost:5173) (or similar port).

## Features

- **Component Playground**: Live demos of `@holmdigital/components`.
- **Convergence Map**: Visual guide to WCAG -> EN 301 549 mapping.
- **Internal Docs**: Internal guidelines for auditing.
